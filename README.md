## 기능
* 할 일 목록(To Do List)
* 시계(Clock)
* 현재 지역 날씨(Weather with Geolocation)
* 사용자 이름 입력

## 학습 내용
* VanillaJS 기초 (Variables, DataType, Object, function 등)
* Local Storage에 데이터 저장학고 불러오기
* Open Weather API 사용하여 계절정보 값 얻어와 사용하기
